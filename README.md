Calculator Project
Welcome to the Calculator Project repository!

This project implements a basic calculator functionality using HTML, CSS, and JavaScript. It allows users to perform arithmetic operations such as addition, subtraction, multiplication, and division.

Usage
Once you have the calculator open in your browser, you can:

Click on the number buttons to input numbers.
Click on the operation buttons (+, -, *, /) to perform operations.
Use the = button to compute the result.
Use the C button to clear the input and start over.
